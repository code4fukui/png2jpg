# png2jpg

a file format converter PNG to JPEG

## usage

```sh
deno -A http://code4fukui.github.io/png2jpg/png2jpg.js image.png
```

```js
import { png2jpg } from "./png2jpg.js";

const jpgbin = png2jpg(pngbin, q);
```
